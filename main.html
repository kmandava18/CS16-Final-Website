About

AP CSA is a class dedicated to learning the basics of programming, specifically in Java. This course involves 12 units, where one is taught how Java's language functions and concludes in an AP exam where this knowledge is put to the test.
There are a couple program one is required to write. An example of this is a project replicating Yahtzee in java.

Final Project

This website is the final project for our class. In the last two weeks after the final, we learned how to code in CSS, JS(JavaScript), and HTML. We basically are learning how to quickly learn programming languages and impliment them. We have, essentially, learned as we've gone and learned from others on the internet.
